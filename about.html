Active Thesis

A Hybrid Ensemble-Based Machine Learning Framework for Electricity Theft Detection in Smart Grids Under Severe Class Imbalance

This research develops an advanced machine learning framework to detect electricity theft in Kenya's power grid through smart meter analytics. The framework addresses the critical challenge of severe class imbalance in real-world State Grid Corporation of China (SGCC) datasets by combining ensemble methods with sophisticated sampling strategies.
