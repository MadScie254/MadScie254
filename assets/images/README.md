# Images Folder

Add your images here:

## Profile
- `profile.jpg` - Your professional headshot (400x400px recommended)

## Projects
Place project screenshots/thumbnails in the `projects/` folder:
- `investwise.jpg`
- `plant-disease.jpg`
- `gdp-prediction.jpg`
etc.

## Icons
Custom icons and favicons go in the `icons/` folder

## Optimization Tips
- Use WebP format for better compression
- Optimize images before uploading (use tools like TinyPNG)
- Recommended max width: 1920px
- Keep file sizes under 200KB when possible
